## 🎯 CORE PRODUCT GOAL

Build a review platform where businesses are rated using a TRAFFIC LIGHT SYSTEM instead of only star ratings:

* Red → High risk
* Orange → Caution
* Green → Strong trust signal

Traffic light is NOT manual review reading. It is computed using rules and signals.

## 📦 CORE FEATURES (MVP)

1. BUSINESS PROFILES

* Each business has:

  * id
  * domain
  * name
  * description (from meta scraping)
  * traffic_light (red/orange/green)
  * green_insurance_eligible (boolean)
* Public page: /business/:id OR /review/:domain

---

2. REVIEWS SYSTEM

* Users can:

  * Post review (rating + text)
  * Edit review
* Review lifecycle:

  * Submitted → moderation queue → approved/rejected
* Only approved reviews appear publicly

3. TRAFFIC LIGHT ENGINE (IMPORTANT)

* Automatically computed using:
